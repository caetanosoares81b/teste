AULA DE HTML

Principais características do HTML É uma linguagem de código de marcação simples É agnóstico em termos de plataforma Permite a construção de diversos elementos, como títulos, parágrafos, links, botões, imagens, listas, tabelas, formulários, vídeos e áudios Permite a interação com outras linguagens, como CSS e JavaScript

História do HTML

A primeira versão do HTML foi publicada em 1991 Foi criada pelo Físico Britânico Tim Berners-Lee O objetivo era facilitar a comunicação e disseminação de documentos entre pesquisadores

Como funciona o HTML

Utiliza marcadores (tags) e atributos para definir como o conteúdo deve ser apresentado num navegador web As tags são identificadas com um sinal de menor que (<), seguido pelo nome da tag e um sinal de maior que (>) A maioria das tags precisa ser fechada, o que representamos com uma barra (/) entre o sinal “<“ e o nome da tag
